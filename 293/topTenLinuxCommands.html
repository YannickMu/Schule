1. $ htop

    Der htop command ist eine schöne version des top commands und zeigt wie der top Befehl auch die Prozesse und ihren Leistungsverbrauch auf.

  2. $ sudo

    Wenn der eigene Benutzer oder eine Gruppe in der man ist in der /etc/sudoers Datei steht kann man die Befehle welche einem gestattet wurden als root ausführen.
    Somit ist man mit dem sudo Befehl allmächtig.

  3. $ ls

    Mit dem ls Befehl kann man alle Dateien in einem Verzeichniss auflisten und mit der option -l auch deren Berechtigungen anzeigen.

  4. $ top

    Mit dem top Befehl können die laufenden Prozesse angezeigt werden, welche am meisten Leistung verbrauchen.
    Wie der Task-Manager unter Windows.

  5. $ grep

    Mit dem Befehl grep kann man die Ausgabe ganz einfach auf die Dinge beschränken, welche man auch wirklich braucht.
    Es ist also eine Art Suchfunktion.

  6. $ more

    Mit dem more Befehl kann man eine Ausgabe auf eine Seite beschränken und somit von oben bis unten durchzulesen, da man ohne GUI im Terminal nicht scrollen kann.

  7. $ vim

    vim ist ein sehr einfacher Texteditor, welcher aber stark personalisierbar ist, der Vorteil von vim ist er ist bei fast jeder Linux-Distro vorinstalliert.

  8. $ cat

    Mit dem cat Befehl kann man sehr einfach eine Datei ausgeben.

  9. $ date

    Der date Befehl gibt das aktuelle Datum und die aktuelle Uhrzeit zurück.

  10. $ echo

    Mit dem echo Befehl kann man Texte auf der Konsole ausgeben oder sie in eine Datei umleiten.

$ echo "Meine Top 10 Linux-Commands"
